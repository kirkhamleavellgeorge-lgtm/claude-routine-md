# M3 加减仓策略（每月回顾）

> **一句话骨架：** 周线主开关定方向（战略层），日线二级阶梯定节奏（战术层），溢价只做薄覆盖层（卖出侧给红利、买入侧设闸门）。
> **整套策略是一个大周期的四个阶段：建仓 → 持仓 → 上涨处理 → 下跌处理**，首尾相接成闭环：**低溢价建仓 → 高溢价收割 → 压缩再建**。先读 M3.0 骨架（贯穿四阶段），再读四个阶段，最后是横切规则（M3.5）。

## M3.0 通用骨架（贯穿四阶段，所有动作都受它约束）

### 仓位四档映射 + 两个时钟

| 信号状态 | 目标仓位 | 时钟（关键区别） |
|---|---|---|
| 周线没走坏（总开关 on） | 守 / 补回 **25% 底仓** | **机械分批**：不择时、不等信号、缺了就补 |
| 日线确认走好 | 加至 **50%** | **确认触发**：等信号才动 |
| 周线确认突破（≠没走坏） | 加至 **75%**（强趋势续 100%） | **确认触发**：等信号才动 |
| 周线走坏（总开关 off） | 释放底仓、退出 | 总开关 |

- **双时钟铁律：** 底仓走"机械分批"的钟，阶梯走"确认触发"的钟。**别用确认时钟补地面**（等周线/溢价都完美 → 永远追不上 → 踏空），**别用机械时钟冲阶梯**（变追高 / 越权动底仓）。
- **周线判定口径（5 周 / 10 周均线交叉 + 周线 MACD 共同判断）：**
  - **走好** = 5 周位于 / 上穿 10 周（金叉·多头排列）+ MACD 同向（DIF>DEA 或红柱走强）。
  - **走坏** = 5 周下穿 10 周（死叉）+ MACD **0 轴下方**死叉 / 绿柱走强 + 价格**有效跌破**关键周线支撑。
  - **钝化 / 模糊期** = 开口收窄、均线钝化，或 MACD 0 轴上方金叉死叉交替（小柱体）→ **默认趋势延续、维持底仓不动**，举证责任落在"走坏"一方。
- **"周线没走坏" ≠ "周线确认突破"：** 前者是守底仓的**被动条件**（钝化时仍成立）；后者是上 75% 的**主动信号**（5W/10W 重新张口向上 + MACD 0 轴上方放量金叉 / 破前高）。**混用 = 把修复当突破、把反弹当趋势 → 追高。**

### 🔒 底仓铁律（整套策略的地基）

> **25% 底仓神圣，唯一释放钥匙 = 周均线走坏。任何溢价水平（即使 >12%）、任何恐慌、任何相机决策都不得击穿。**

底仓有双重身份：① 战略配置；② **操作使能器**——
- **有底仓** = 已吃住反弹第一段、均价锚在低位 → 阶梯可机械执行、不必追高。
- **无底仓** = 每次从零起 + 必在涨幅后才确认 → 买入 = 高价 × 高溢价双杀 → 买不下手 → **系统性踏空（M4.3 恶性循环根源）**。
- **守底仓 = 守整条阶梯的可执行性。** 这是"神圣"的操作层理由，独立于且强于战略配置理由。

### 溢价的角色（不对称：帮卖、坑买；一切盈亏按 ETF 口径）

| | 溢价帮你 | 溢价坑你 |
|---|---|---|
| **卖出侧** | 高溢价卖 = 出场红利 | 压缩溢价里被迫卖（痛苦） |
| **买入侧** | 上涨杀溢价买 = 入场红利 | 拉伸溢价里追买（高位回归风险） |

- **价↑溢价↓（上涨杀溢价）**：净值涨 > 场内价涨，溢价被吸收（例：旧 6%、美股 +3.5/场内 +1.6 → 新 ≈ +4%，未经历下跌）= **最佳加仓窗口**。
- **价↑溢价↑（双拉伸）**：危险追高，**不加战术仓**（仅底仓被动参与）。
- **价↓溢价↓（下跌杀溢价）**：场内价跌向净值、常伴回撤，痛苦；仅"趋势明确许可 + 到 regime 下沿"时作次优补点。
- **ETF 口径（铁律）：** 一切盈亏、懊悔、加减仓决策按场内 ETF 价（含溢价）衡量，**不按纳指口径**（M4.3 教训）。

## M3.1 建仓（空仓 / 低仓 → 25% 底仓 → 启动 50% 阶梯）

> 对应循环的 ⓪筑底 → ①反弹初起 → ②反弹确认。**当前你正处于此段（1% 仓位、99% 现金、待重建）。**

**指数应对：**
- 周线没走坏 → **机械分批 / 网格回补 25% 底仓**：不择时、不等信号、不追单一最优点位（"买得不够便宜"可接受，目标是恢复战略配置、避免踏空，不是抄底）。
- 日线确认走好（有效突破并站稳）→ 加至 **50%**。
- **钝化期日线作方向先行信号**：周线读不出方向时，看日线能否**有效突破**——突破站稳 → 周线大概率重新张口向上 → 阶梯可推进；持续不破 + 周线收口 → 转入周线级调整 → 守地面、不加战术仓。

**溢价应对：**
- **底仓回补不受 <5% 溢价闸门约束**（战略层、由周线开关驱动），但事件不操作窗口仍规避追入。
- **战术仓（50/75）才看溢价**：冷启动 / 战术重建闸门 = **溢价 < 5%**；优选"价↑溢价↓"（上涨杀溢价）窗口下手，溢价给安全垫；**"价↑溢价↑"绝不追**。
- 第一档战术加仓的"提前变体"仅作**"限额 × 真压缩（价↑溢价↓确凿）"的例外**，非常规规则（防 M4.3 越权）。

**❌ 不这样做的危害：**
- 用"确认时钟"去补底仓（等周线 / 溢价都完美才买）→ 永远追不上一轮上涨 → **系统性踏空**（M4.3：砍到 1% 后纳指连涨）。
- 在拉伸溢价里追战术仓 → 高价 × 高溢价双杀 → 浮亏 + 心理压力 → 下次更买不下手 → 恶性循环。

## M3.2 持仓（趋势确立 → 加至 75/100 → 高位维持）

> 对应循环的 ③趋势确立 → ④高位钝化。

**指数应对：**
- 周线**确认突破**（5W/10W 重新张口向上 + MACD 0 轴上放量金叉 / 破前高，**≠ 仅"没走坏"**）→ 加至 **75%**，常规强趋势可续至 **100%**。
- 持有期持续盯周线开关：只要没走坏就持有 / 在位；**周线末期上限降至 75%**。

**溢价应对：**
- 溢价进上区（结构性抬升）→ **改粗阶梯 / 小批加**，**别在拉伸溢价里追满**。
- ④高位钝化（满仓 + 溢价 7–8% + 周线仍上但动力衰减）→ **停止加仓 + 启动卖出侧溢价收割**（转入 M3.3）。

**❌ 不这样做的危害：**
- 把"修复 / 反弹"当"突破"加到 75–100% → 追高，反弹一结束即套牢。
- 满仓后忽略溢价累积（7–8% 仍加）→ 顶部一旦回调 = **指数 + 溢价双杀**（M4.2 标注的最大尾部心理风险，合计 10%+ 尚未经历）。

## M3.3 上涨处理（高位减仓 + 溢价收割）

> 这是整圈里**唯一"溢价帮你"的半程**——把虚高的场内价兑现掉。所有动作均**在 25% 底仓之上**操作。

**指数应对（价格驱动减仓·一级预防）：**
- 涨幅充分 + 日线走差 → 减至 **50%**。

**溢价应对（卖出侧红利）：**
- 溢价 **>10% → 减 5%；>12% → 减 10%**。
- ④高位钝化（满仓 + 溢价 7–8% + 周线动力衰减）→ 主动启动收割。

**宏观 / 政策卖出覆盖（叠加在上）：**
- 纳指破前高 + VIX<14 → −5%；10Y 破 4.8% → −5%；单日股债同跌 1.5%+ → 先减 3%。
- QDII 额度放开 → 立即减 5%；监管警示溢价 → 评估减 5–10%；基金恢复 / 扩大申购 → 减 3–5%。

**❌ 不这样做的危害：**
- 不在高溢价兑现 → 错过整个周期里唯一"溢价帮你"的窗口；等溢价被压缩时才卖 = 在**压缩溢价里被迫卖**（痛苦卖、亏掉本可兑现的溢价红利）。

## M3.4 下跌处理（防守阶梯 + 底仓铁律 + 机会性逆势）

**指数应对（价格驱动减仓·二级 / 终极）：**
- 跌破日 20MA → 减至 **25%（底仓）**。
- 周均线走坏（**唯一释放钥匙**）→ 跌破底仓，清仓 / 留最小观察仓。

**溢价应对：**
- 下跌杀溢价（场内价跌向净值、常伴回撤）= 痛苦段，**本身不是加仓理由**。
- **机会性逆势加仓（非主引擎）：** 周线仍向好却闪崩、纳指急挫 **−10% / −15% 且溢价 <3%** → 机会性补点（趋势许可 + regime 下沿）。

**🔒 铁律复述（下跌段最易被破，单列强调）：**
> **任何溢价水平（即使 >12%）都不得击穿 25% 底仓；唯一释放钥匙 = 周线走坏，不是溢价、不是恐慌、不是"溢价太高"。**

**❌ 不这样做的危害（这是你已踩过的坑）：**
- 用溢价顾虑越权砍底仓（把"溢价太高"当减仓理由砍穿 25%）→ 击穿后纳指连涨 = **踏空**（M4.3：2026-06-09/10 从 50%→25%→1%）。
- 底仓一旦没了 → 整条阶梯失去可执行性（无底仓 = 从零追高 = 买不下手）→ 把一次踏空变成**系统性踏空**。

## M3.5 横切规则（贯穿四阶段，集中归并、避免散落）

### A. 周线钝化 / 失真过滤（长涨后 + 震荡期，降低单一指标权重、提高确认门槛）
1. **柱体阈值：** 看 MACD 红绿柱体大小 / 面积，**小柱体交叉 = 噪音、不触发动作**（不以 DIF/DEA 交叉本身为准）。
2. **0 轴定权：** 0 轴上方死叉多为洗盘 / 钝化（降权、倾向无视）；**0 轴下方死叉才赋"走坏"权重**。
3. **价格结构优先：** 钝化期均线 / MACD 滞后，改以价格结构为准——周线 higher-low 是否成立、关键周均线是否**有效跌破并站稳**（收周线确认、连续 2 周不反复）。
4. **月线终极过滤：** 月线为最高级开关；**月线未坏 → 周线钝化期不必转空、维持底仓**。

### C. 阶梯 vs 一段式·回撤测算（校准 M3.1–M3.4 的分级粒度）

可复算模型：组合净值（每段）= 上段净值 ×（1 + 仓位比例 × 该段指数涨跌幅）；触发点切换目标档；最大利润回撤 = 净值峰值到谷值最大跌幅。

| 价格路径 | 2 级·最大回撤 | 1 段式·最大回撤 | 2 级·最终收益 | 1 段式·最终收益 |
|---|---|---|---|---|
| 温和回调 | −3.9% | −5.5% | +19.2% | +18.9% |
| 深度回调 | −6.5% | −9.4% | +22.2% | +20.4% |
| 急速 V 反转 | −2.7% | −3.6% | +20.6% | +21.7% |

> **固化结论：** 2 级在任何路径都少回撤 **30–40%**（回撤 / 情绪平滑器，契合 M4.2/M4.3）；慢调 / 深调收益 ≥ 1 段式，**仅急速 V 反转落后约 1%**。**溢价修正必计**：2 级多 1 次加回，踩溢价回归按 ~1–2%/次拖累，会吃掉慢调 / 深调的微弱优势、加重急 V 落后。**决策含义：把 2 级当回撤管理工具，不当增厚收益工具——预判震荡 / 深调用 2 级；预判单边急涨 / 急 V 简化为 1 段式或守底仓不动。**

# M4 投资观点与心理边界（季度 ~ 半年回顾）

## M4.1 主导观点

1. 美股长周期偏多（赌 AI 资本支出周期 + 长期向好）
2. 下跌形态偏"震荡做周线头 → 压合周均线 → 再突破"，而非 V 型崩盘
3. 6 月偏震荡（高位 + 外部动荡，推高动力不足）；鸽派概率偏高（赌特朗普主动制造催化剂）
4. 对溢价回归风险敏感；理解 EIA 库存的领先意义

**关注优先级（★★★★★）：** ETF 溢价 + QDII 政策、纳指 100、油价/地缘、通胀/美联储。次级：VIX、美元/利率、标普。

## M4.2 心理边界

| 边界 | 状态 |
|------|------|
| 浮盈回吐到 0% | 边缘（可能情绪化决策） |
| 浮盈回吐 50% | 可承受 |
| 单日溢价收窄 1–2% | 已经历，可承受 |
| 单日溢价收窄 5%+ | 未经历，存在心理冲击 |

**决策偏好：** 倾向"稳定主仓 + 分批加仓"、接受小幅减仓锁利、重纪律避免决策疲劳；不做"先减后加"（自评难控）、不做短线/频繁操作。

## M4.3 已踩坑教训

- **三步减仓 50%→25%→1%（2026-06-09/10）击穿了 25% 底仓。** 溢价顾虑（记忆 >10%，日志 6.2%，存疑）越权动了本应只归周线管的底仓，事后纳指连涨踏空。教训：①"底仓神圣"此前未写入、压力下最先被破，现已显式写入 M3.1；②用 ETF 口径（场内当日 +1.5%）而非纳指口径（+3.5%）衡量盈亏与懊悔。
- 待补强的盲点：对单一二手数据点的警惕、对鸽派情景持续性的可能高估。

# M5 基本盘与沟通约束（极少变动）

## M5.1 背景

中长线投资者（看周线/月线），通过中国大陆 QDII 类 ETF 投资美股；有经验、已建立独立分析框架；风险承受中等。

## M5.2 红线（绝不建议的方向）

杠杆/期权/衍生品激进策略、单一个股重仓、短线/日内、加密货币具体操作、A 股/港股个股推荐（港股 ADR 参考除外）、QDII 套利（不博溢价价差）。

## M5.3 沟通约束（PRD 已含全局规则，此处仅记用户偏好）

- 风格：数据 + 概率 + 触发点三层；带反共识视角；明确区分"事实/推断"；中文结构化（表格 + 分级标题）；避免情绪化词与冗长铺垫。
- 回应方式：用户提判断时先肯定对的部分再压力测试；表达情绪时识别但不放大、回到数据；数字存疑时给区间 + 来源差异。
- 强约束：不替用户决策、不预测精确点位/时点、每次标注最大"已知未知"、所有数字带时间戳、不忽视溢价维度、区分美股事件 vs ETF 事件。
